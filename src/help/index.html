Search:

  • Click the "Search" Button or "Alt+S" to search for the word you typed.(or)
  • Press the "Return" key "double times" to search for the word you typed.(or)
  • "Double Click" a word in the Suggested words list to search.(or)
  • "Double Click" a word on the side docks("History", "Book Marks") to search.

File Menu

Add a new word to the dictionary which is not available in the dictionary. Click save button only both fields are filled otherwise Click "Cancel" button if you want to cancel the entry. Use special type writers for Tamil Unicode such as Goolge Input, eKalappai, etc.,

Increase the font size within the limit.
Decrease the font size within the limit.
Options Change the default behaviour of the applications. You can set to close the dock windows(History, Book Mark) permanently; You can delete all the past history of word search from the beginning. The changes will be applied on the next application start-up or you can revert the changes by clicking the "Set to defaults" option.
